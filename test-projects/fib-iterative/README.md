# fib-iterative

An end-to-end test project for the Opalescent compiler. This project computes the 10th Fibonacci number using an iterative algorithm to verify mutable variables, while loops, and string interpolation.

## How to compile and run

Use the Opalescent compiler to compile and execute the program:

```bash
opal src/main.op
```

This will print `55` to stdout (the 10th Fibonacci number).
