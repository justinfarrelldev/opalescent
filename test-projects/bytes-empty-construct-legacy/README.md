# bytes-empty-construct-legacy

An end-to-end test project for the legacy `bytes_new()` constructor. It verifies that an empty Bytes buffer compiles, runs, and reports a length of zero.

## How to compile and run

Use the Opalescent compiler to compile and execute the program:

```bash
opal src/main.op
```

This will print `legacy length: 0` to stdout.
