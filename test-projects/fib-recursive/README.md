# fib-recursive

An end-to-end test project for the Opalescent compiler. This project computes the 10th Fibonacci number using a recursive algorithm to verify function recursion, integer arithmetic, and string interpolation.

## How to compile and run

Use the Opalescent compiler to compile and execute the program:

```bash
opal src/main.op
```

This will print `55` to stdout (the 10th Fibonacci number).
